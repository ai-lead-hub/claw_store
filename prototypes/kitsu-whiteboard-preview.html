Prototype preview / kitsu + whiteboard architecture

Keep the generation lab, add project memory and collaborative review.

This prototype extends your Stitch generation UI without forcing the whole product into a giant infinite canvas. Kitsu stays the production backbone. The tool becomes a creative operating layer with three main surfaces: a strong generation panel, a reusable asset browser filtered by shot context, and a collaborative whiteboard linked to publishes and comments.

What this preview is testing

  • Kitsu provides project and shot context, but does not hard-lock the UI into rigid shot workspaces.
  • Shot number acts like a contextual filter for assets, versions, and things needing attention.
  • The generation experience still feels like the polished dark “lab” you explored in Stitch.
  • Collaboration lives in a spatial whiteboard instead of burying comments in side panels.
