## Mandatory source tiers per entity

| Tier | Sources |
|---|---|
| Official | blog, product page, release notes, GitHub, arXiv |
| Video | YouTube (official + 3+ creators/analysts), Bilibili for Chinese labs |
| Community | Reddit, X/Twitter search, HN, Discord |
| News | 2+ tech/news publications per high-priority entity |

**One source type is failure. Cover all four.**

## Importance ranking

Rank by:
1. direct creative utility
2. startup relevance
3. workflow leverage
4. proof quality

Prefer shipped reality over hype.

## Daily output

Create or update the relevant daily intake file under `intel/daily/`.

Each captured item should have:
- title
- source
- date
- link
- category
- tags
- quick note
- rough importance
- summary status
- promotion target

## Promotion guidance

Promote strong items into:
- `intel/urgent/`
- `intel/workflow-opportunities/`
- `intel/research-papers/`
- `intel/entities/`
- `intel/experiments/`

Reports are outputs, but the durable knowledge base is the real asset.
