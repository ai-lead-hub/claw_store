# Implementation Plan

## Objective
Create a research operating system for AI filmmaking intelligence that compounds into reusable company knowledge.

## Phase 1 — Foundation

Deliverables:
- repo structure
- templates
- taxonomy
- scoring model
- source watchlists
- cron/scheduling plan

Success criteria:
- intake is easy
- reporting is consistent
- durable notes have clear homes

## Phase 2 — Operating loop

Deliverables:
- daily collection workflow
- every-other-day report workflow
- promotion rules for important items
- basic dedupe process across recent days

Success criteria:
- useful raw intake every day
- clear high-priority shortlist
- reports are strategic, not generic

## Phase 3 — Intelligence compounding

Deliverables:
- entity notes for major labs/tools/creators/repos
- workflow opportunity notes
- paper notes with implementation implications
- experiment queue

Success criteria:
- recurring signal accumulates in reusable notes
- research begins influencing roadmap and testing

## Phase 4 — Optional automation upgrades

Possible additions:
- JSONL sidecars for structured retrieval
- source-specific fetch helpers
- automated dedupe suggestions
- scoring assistance
- report assembly helpers

## Working principles

- keep v1 simple enough to use
- keep structure strong enough to scale
- bias toward actionable signal
- promote only what is worth re-reading later
